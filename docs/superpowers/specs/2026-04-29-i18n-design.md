# i18n — Vive Gostoso

**Date:** 2026-04-29  
**Status:** Approved — implementing

## Overview

Add PT / EN / ES language support to all public-facing pages. Admin and cadastre pages stay in Portuguese. Language stored in localStorage (`vg-lang`). Dark-mode toggle moves from header to footer.

## Stack

- `i18next` + `react-i18next` + `i18next-browser-languagedetector`
- Detection order: `localStorage` → `navigator` → fallback `pt`
- localStorage key: `vg-lang`

## Architecture

```
src/
  i18n.ts                          # i18next init
  locales/
    pt.json
    en.json
    es.json
  components/
    i18n/
      language-selector.tsx        # Globe + sigla pill (PT | EN | ES)
  components/layout/
    header.tsx                     # +LanguageSelector, -ThemeToggle
    footer.tsx                     # +ThemeToggle (sun/moon, discrete)
```

## LanguageSelector

- Placement: header right side, before the login/Negócios buttons
- Visual: globe icon (Lucide `Globe`) + current language code (PT / EN / ES)
- Dropdown: three options, subtle border pill, matches existing rounded-full style
- Colors: uses existing `text-[#3D3D3D]`, `hover:bg-areia`, `teal` accent for active
- Mobile drawer: language selector at bottom of drawer, same style

## ThemeToggle in Footer

- Bottom-right corner of footer bottom bar (next to "Feito pela Balaio" link)
- Small sun/moon icon, opacity-60 default, hover opacity-100
- White color to match footer palette

## Scope

**Translated (public):** Home, Come, Fique, Passeie, Explore, Participe, Contrate, Servicos, Sobre, Conheca, Apoie, Negocio, Evento, NotFound, Header, Footer

**Not translated (stays PT):** All `/cadastre/*` pages (admin/business owner area)

## Locale file structure

Keys are namespaced by section: `nav.*`, `home.*`, `common.*`, `come.*`, `footer.*`, etc.

## Design constraints

- Language selector must not look foreign to the design — same font, same border style
- ThemeToggle in footer: white icon, matches footer color scheme
- No layout shift on language change (all strings same approximate length)
